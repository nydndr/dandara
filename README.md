# dandara

This is my portfolio website implemented on Next.js. It contains various writing about the projects I've developed and a weekly documentation of my learnings, finding and likings on Development and Design.

This is an always ongoing project, I'm always redesigning, polishing or using it to learn new technologies.

## Overview

-   `pages/journal` - [Work Journal](http://dandara.vercel.app/journal) weekly writings about everything I did, learned and liked about the development and design world.
-   `content/entries` - Markdown files that feed the journal page, using [gray-matter](https://www.npmjs.com/package/gray-matter) and [react-markdown](https://github.com/remarkjs/react-markdown).
-   `pages/*` - All other static pages.

## Getting Started

In case you wanna run the project - use it as a starting point or just dive in the code:

```bash
$ git clone https://github.com/nydndr/dandara.git
$ cd dandara
$ npm install
$ npm run dev
```

## Built With

-   [Next.js](https://nextjs.org/) - React framework of choice
-   [Vercel](https://vercel.com) - For Next.js deployment
-   [Tailwind CSS](https://tailwindcss.com/) - For utility-first CSS
-   [React Markdown](https://github.com/remarkjs/react-markdown) - For markdown rendering
-   [GSAP](https://greensock.com/gsap/) - For perfomant animations

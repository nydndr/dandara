# dandara

Esse é meu portfólio, implementado em `Next.js`. Ele contem meu portfólio, várias escritas sobre projetos que eu desenvolvi e uma documentação semanal de tudo que eu aprendi, encontrei e curti sobre o mundo do design e do desenvolvimento.

Esse é um projeto em constante evolução. Estou sempre trabalhando em um re-design, polindo algumas coisas e utilizando ele pra aprender novas tecnologias.

## 101

-   `pages/journal` - [Journal de desenvolvimento](http://dandara.vercel.app/journal) escritas semanais sobre tudo que eu fiz, aprendi e curti sobre o mundo do design e do desenvolvimento.
-   `content/entries` - Os arquivos markdown que alimental o journal usando `[gray-matter](https://www.npmjs.com/package/gray-matter)` e `[react-markdown](https://github.com/remarkjs/react-markdown)`.
-   `pages/*` - Todas as páginas estáticas que o `next.js` toma conta.

## Começando

Caso você queira rodar o projeto, usá-lo como ponto de partida ou só mergulhar no código:

```bash
$ git clone https://github.com/nydndr/dandara.git
$ cd dandara
$ npm install
$ npm run dev
```

## Construído com

-   [Next.js](https://nextjs.org/) - Framework React de escolha
-   [Vercel](https://vercel.com) - Melhor experiência de deploy para o Next.js
-   [Tailwind CSS](https://tailwindcss.com/) - Para CSS na modalidade utility-first class
-   [React Markdown](https://github.com/remarkjs/react-markdown) - Para renderização de Markdown
-   [GSAP](https://greensock.com/gsap/) - Para animações perfomáticas

## Futuro

Esse website se encontra em fase de redesign. A seção em desenvolvimento no momento sempre possui uma branch ativa, no `readme.md` dela você pode acompanhar as próximas implementações.
